# 📝 React Notes App

A simple and clean Notes App built using React. Users can add and delete notes with a minimal UI.

---

## 🚀 Features

* ➕ Add notes with title and description
* ❌ Delete notes instantly
* ⚡ Real-time UI updates using React state
* 🎨 Clean and responsive UI (Tailwind CSS)

---

## 🛠️ Tech Stack

* React (useState Hook)
* Tailwind CSS
* Lucide Icons



## 📂 Folder Structure

src/
├── App.jsx
├── main.jsx
└── index.css

---

## ⚙️ How to Run Locally

```bash
git clone https://github.com/Hardboy007/react-notes-app.git
cd react-notes-app
npm install
npm run dev
```
